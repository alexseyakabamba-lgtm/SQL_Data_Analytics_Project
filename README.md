# SQL Server Data Analytics Project

## Project Overview

This project was developed to build, demonstrate, and showcase my SQL data analytics skills through a complete end-to-end analytical workflow.

Using the Gold layer generated by my Data Warehouse project, I perform comprehensive Exploratory Data Analysis (EDA), progressing from basic database exploration to advanced customer and product reporting. Throughout the project, I apply a wide range of SQL techniques to transform business data into meaningful insights and decision-ready reports.

## Project Purpose

Designed as a portfolio project, it demonstrates my ability to solve real-world business problems using SQL while showcasing best practices in writing clean, modular, and well-documented code. views for business intelligence
Write clean, maintainable, production-style SQL scripts

## Structure of the project

The project is organized into modular SQL scripts that cover both Exploratory Data Analysis (EDA) and Advanced Analytics, including database and dimension exploration, KPI analysis, ranking, trend analysis, cumulative analysis, customer and product segmentation, performance analysis, part-to-whole analysis, and the creation of business reporting views.
